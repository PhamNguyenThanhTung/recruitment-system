Đồ án môn Công Nghệ Phần Mềm — Nhóm [số nhóm]

## 🛠️ Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS, Zod
- **Backend:** Next.js API Routes, NextAuth.js
- **Database:** MongoDB Atlas / PostgreSQL
- **Deploy:** Vercel
- **File Upload:** Cloudinary

## 🚀 Cách chạy dự án
```bash
# 1. Clone repo
git clone https://github.com/PhamNguyenThanhTung/recruitment-system

# 2. Cài dependencies
npm install

# 3. Tạo file môi trường
cp .env.example .env.local
# Điền các biến vào .env.local

# 4. Chạy dev server
npm run dev
```

## 🌿 Quy tắc nhánh
- `main` — nhánh chính, chỉ merge qua PR
- `feature/[tên-tính-năng]` — nhánh tính năng
- Ví dụ: `feature/auth-hr`, `feature/job-crud`
